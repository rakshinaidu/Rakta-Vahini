# Rakta-Vahini : Smart Blood Donor Management System

## Overview
Rakta-Vahini is an Android application developed using Kotlin and Firebase to help users register as blood donors, search donors based on blood group and city, track donations, and manage emergency blood requests efficiently.

---

## Features
- Firebase Authentication
- Donor Registration
- Blood Group Search
- Donation Logging
- Statistics Dashboard
- Firebase Firestore Integration

---

## Technologies Used
- Kotlin
- Android Studio
- Firebase
- Firestore
- XML
- GitHub
